# GazelBooks MVP (Static Vercel)

Ini adalah MVP GazelBooks versi static (1 file) yang bisa langsung di-deploy ke Vercel.

## Cara Deploy (paling gampang)
1. Download ZIP ini
2. Extract
3. Upload folder ke GitHub
4. Import project di Vercel
5. Deploy

- Cocok untuk demo / MVP cepat
